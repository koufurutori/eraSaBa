# eraSaBa
◇◇위 게임에 등장하는 모든 인물들은 성인이며, 단순히 극에 맞춰 연기를 하고 있을 뿐입니다!◇◇

「마법소녀의 마녀재판」의 2차 창작 비공식 팬게임인 에라사바(eraSaBA)입니다.<br />
eraTWKR 4.910-KR을 기반으로 개조+가필하여 만들었습니다.<br />
위 게임은 무료 배포며, 이곳에서만 배포하고 있습니다. 혹 다른 곳에서 파일을 댓가로 금전을 요구하거나 재배포하는 것을 발견하시면 경추(耕隹/koufurutori)에게 알려주세요.

era게임 특성상 개인에 따라 민감하다고 여길 수 있는 부분이 있습니다.<br />
또한 개개인의 구상에는 라이센스가 부여됩니다. 원작자의 동의없이 수정하지 마세요.


제가 작성한 부분(개조판에서 추가된 부분)에 대해서는 다음과 같은 행위를 허용합니다.
- 3차 창작
- 버그, 오탈자 수정

이하의 행위에 대해서는 연락을 우선 주세요.
- 구상에 가필, 편집, 혹은 기타 개변을 하는 경우
- 타 era 개조판에서 사용하는 경우
- era 이외의 환경에서 사용하는 경우

이하의 행위는 금지합니다.
- 상업적 행위 및 금전적 이익을 위한 이용
- 구상 작성자의 이름을 지우는 행위


이 게임을 플레이하거나 사용한 것으로 인해 생기는 손해에 대해서 제작자는 일체의 책임을 지지 않습니다.
